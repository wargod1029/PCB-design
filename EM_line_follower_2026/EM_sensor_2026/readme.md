# EM_sensor

Brief
- EM sensor board designed to measure magnetic field generated by copper wire with AC signal (20kHz)

Key features
- use LC circuit to detect the AC signal on copper wire in 20kHz and send it to the mainboard by connector

Repository structure
- `ken_EMstick.kicad_pcb` — PCB layout
- `ken_EMstick.kicad_sch` — Schematic
- `ken_EMstick.kicad_pro` — KiCad project file
- `ibom.html` — HTML BOM export

Connectors & pins
- POWER: GND (do not need VCC)
- SENSOR: L1 L2 L3 L4 L5

Fabrication notes
- PCB thickness: 1.6 mm
- Layer count: 2
- Copper thickness: 1 oz external

BOM
- Use KiCad's BOM export (`bom` plugin) to generate an initial parts list.

KiCad / development notes
- KiCad version used: 8.0
