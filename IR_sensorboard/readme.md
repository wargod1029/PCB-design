# IR_sensorboard

Brief
- IR sensor board designed for line-following applications

Key features
- Infrared sensing 

Repository structure
- `netcar.kicad_pcb` — PCB layout
- `netcar.kicad_sch` — Schematic
- `netcar.kicad_pro` — KiCad project file
- `netcar.step` — 3D model export

Electrical overview (fill with measured values)
- Supply voltage: 3.3V - 5V

Connectors & pins
- POWER: VCC, GND
- SENSOR: IR signal outputs by SPI

Fabrication notes
- PCB thickness: 1.6 mm
- Layer count: 2
- Copper thickness: 1 oz external

BOM
- Use KiCad's BOM export (`bom` plugin)


KiCad / development notes
- KiCad version used: 8.0
